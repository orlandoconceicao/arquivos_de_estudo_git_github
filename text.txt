Sequencias fundamentais:

- abrir github 
- ir em repositories 
- new em verde 
- colocar nome e observação 
- fazer oque o github manda

- selecionar pasta                 | cd  .\arquivo\
- limpar terminal                  | clear
- adicionar git no vscode\terminal | git init
- adicionar arquivos específicos   | git add (arquivo) (pasta\arquivo)
- adicionar todos arquivos         | git add .                              |Mesmo dentro de pasta
- salvar todas mudanças            | git commit -a -m "mensagem"
- salvar mudanças epecíficas       | git commit (arquivo) -m "mensagem"
- verificar informações            | git status                             |Usar frequentemente para nao perder nada
- enviar arquivos para o repos     | git push                               
- abaixa e mescla e salva o branch | git pull                               |usar frequentemente para ver si nao ha nada novo  | Pega atualizações tbm
- clonar repositórios              | git clone (url) .                      |ir no repositório ir em code/copiar url
- remover arquivo                  | git rm (arquivo)
- puxar histórico de atualizações  | git log                                |mostra quem, data e hora                           
- mover para outra pasta           | git mv (1arquivo) (pasta/1arquivo)      
- renomear arquivo                 | git mv (errado) (certo)
